# 📚 Мой Дедлайн — Telegram-бот для студентов

**Мой Дедлайн** — это Telegram-бот, который помогает студентам и школьникам не забывать о заданиях, дедлайнах и учебных делах.  
Проект создан как **индивидуальный учебный проект** и представляет собой удобный инструмент для управления задачами прямо в Telegram.

---

## ✨ Возможности бота

Бот умеет:

- ➕ добавлять новые задачи;
- 📋 показывать список активных и выполненных задач;
- ✅ отмечать задачи как выполненные;
- 🗑 удалять задачи;
- 🔴🟡🟢 задавать приоритет задачи;
- 📅 показывать задачи в календарном виде;
- ⏰ присылать напоминания о дедлайнах;
- ⚙️ менять формат отображения даты;
- 🔔 настраивать режим напоминаний;
- ↩️ возвращаться на предыдущий шаг при добавлении задачи;
- ❌ отменять добавление задачи.

---

## 🎯 Актуальность проекта

Многие студенты сталкиваются с проблемой: задания, домашние работы, подготовка к контрольным и проектам часто записываются в разных местах или не записываются вовсе.  
Из-за этого часть дел забывается, сроки пропускаются, а нагрузка распределяется неравномерно.

Бот **«Мой Дедлайн»** решает эту проблему, помогая:

- держать все задачи в одном месте;
- видеть дедлайны заранее;
- быстро ориентироваться по важности дел;
- получать напоминания без необходимости открывать отдельные приложения.

---

## 🛠 Используемые технологии

Проект реализован с помощью:

- **Python 3**
- **aiogram 3**
- **SQLite**
- **python-dotenv**
